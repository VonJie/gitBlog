# issuesBlog
该项目主要用于个人平时工作中的代码笔记、问题收集，方便日后翻阅查找。

最开始的博客实用 hexo 搭建起来，但是每次都要编写发布觉得很烦。

使用 issues 作为博客，主要是省去了发布流程，而且搜索功能很实用。
